# Noor al-Hidayah — نور الهداية

A Shia Islamic knowledge portal for the followers of the Ahl al-Bayt (عليهم السلام).

## Features
- **Daily reminders** — a Qur'anic verse and an authentic narration that rotate every day.
- **The Maraji' al-Taqleed** and their principal books, with official-site links and a per-Marja' "Search Rulings" option.
- **Foundational books** — the Holy Qur'an, Nahj al-Balagha, Sahifa Sajjadiyya, the Four Books (Al-Kafi, Man la yahduruhu al-Faqih, Tahdhib, Istibsar), Bihar al-Anwar, and Mafatih al-Jinan.
- **Unified search** for authentic narrations in both Arabic and English, with source filtering and highlighting.
- **Federated search** that opens your query in the major scholarly libraries: Tanzil (Qur'an), Thaqalayn.net (hadith), and Al-Islam.org.

## Usage
Open `index.html` in any web browser — it is fully self-contained, no build step required.

> Always verify rulings (aḥkām) with the risāla of your own Marji'.
